# 稳定第一周

:::info[本课目标]
**耗时**：约七个游戏日　·　**完成标志**：能稳定获得饮水与食物，并拥有修理、制作和撤退方案
:::

第一周的核心不是囤满仓库，而是把每天的冒险变成可重复、可恢复的循环。

## 建立据点

选择有储物空间、可控入口、附近水源或烹饪条件的位置。按用途划分物资区：食物、药品、工具、衣物和待处理战利品。固定摆放能显著减少每次出发前的准备时间。

## 补齐基础能力

- **饮水**：容器、稳定水源、加热或净水方式。
- **食物**：先处理易腐物，再建立数日储备。
- **医疗**：止血、消毒、包扎与疼痛管理用品。
- **工具**：根据物品说明收集常用工具品质，不要只看物品名称。
- **移动**：轻装探索，标记安全路线和高风险区域。

## 每次探索前后

### 出发前

确认天气、光照、体力、饮水、伤口和撤退时间。只带本次目标真正需要的装备。

### 返回后

先处理伤口和口渴，再分类物资、修理装备并记录下一次目标。不要在疲劳和疼痛状态下立刻开始长时间制作。

## 何时进入下一阶段

当你能连续几天从据点出发并安全返回，就可以尝试载具、长距离探索、农耕、基地建设或更复杂的制作链。此时失败通常不再来自基础资源，而来自信息不足和风险失控。
