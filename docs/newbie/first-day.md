# 活过第一天

:::info[本课目标]
**耗时**：一个游戏日　·　**完成标志**：在安全位置过夜，拥有饮水、容器、简单武器和止血用品
:::

第一天不是清空城镇的竞速。你的优先级是：**脱离眼前危险 → 处理伤口 → 获得安全饮水 → 准备过夜 → 补充食物**。

## 出门前

- 找一个能装水的容器。
- 准备撬棍、棍棒或其他容易使用的临时武器。
- 带少量绷带或可止血材料，不要把所有物资背在身上。
- 确认出生点是否可以作为临时撤退点。

## 搜刮节奏

1. 从城镇边缘或孤立建筑开始，不要直接进入中心区域。
2. 靠近前先观察；一次只吸引一个敌人。
3. 优先搜索厨房、浴室、车库和药柜。
4. 体力下降、疼痛上升或视野中出现多个威胁时立刻撤离。
5. 把重物集中放在临时据点，而不是一直随身携带。

## 水、食物与伤口

### 饮水

找到水不等于能安全饮用。保留容器，并通过烹饪、净水工具或明确标注为洁净的来源处理饮水。不要因为轻微饥饿而冒险，但口渴会很快影响行动。

### 食物

先吃容易腐坏的食物，罐头和干粮留作储备。阅读物品状态，避免在没有必要时食用腐坏或来源可疑的食物。

### 伤口

流血要立刻处理；疼痛会降低战斗和移动能力。战斗结束后检查每个身体部位，不要只看总体状态。

## 选择过夜点

一个合格的临时据点应满足：入口少、窗帘可关闭、附近没有持续可见的敌人，并且你清楚至少一条撤退路线。睡前把水、药品、武器和照明放在固定位置。

## 如果局面失控

| 情况 | 立即行动 |
|---|---|
| 被多个敌人追赶 | 停止攻击，利用门窗和障碍切断视线 |
| 体力见底 | 改为步行并拉开距离，不再进行新的战斗 |
| 严重流血 | 到掩体后止血，再判断是否继续移动 |
| 天黑还在城内 | 放弃额外搜刮，沿已知路线返回 |
