# 下载与首次启动

:::info[本课目标]
**耗时**：约 10 分钟　·　**完成标志**：看到中文主菜单，并能新建世界
:::

## 下载正确的版本

只从 [CCB GitHub Releases](https://github.com/LYHGLYTX/Cataclysm-Cleanwater-Bomb/releases) 获取正式构建。发布页通常会包含多个平台和架构，请根据设备选择。

| 平台 | 选择与启动方式 |
|---|---|
| Windows | 下载 64 位图形版压缩包，完整解压后运行 `cataclysm-tiles.exe` |
| Linux | 下载对应发行包，或按照[编译教程](/docs/dev/build)自行构建 |
| Android | 下载 APK；系统提示时仅授权浏览器或文件管理器安装此文件 |
| macOS | 使用发布页提供的包；首次启动可能需要在系统安全设置中确认 |

:::warning[不要在压缩包里直接运行]
游戏需要读取同目录的数据、字体和贴图。Windows 用户必须先把整个压缩包解压到可写目录。
:::

## 第一次启动

进入 **选项** 后先调整以下项目，选项名称可能随上游同步略有变化：

1. 在常规设置中选择简体中文，然后重启游戏。
2. 在图形设置中启用贴图，并选择随版本提供的贴图包。
3. 根据屏幕调节终端宽高、字体大小与缩放。
4. 打开自动保存；保留默认按键，等熟悉后再改。

## 创建练习世界

第一次不要堆叠大量 MOD。创建世界时保持接近默认配置，角色选择普通的避难所开局，把目标定为“熟悉操作”，而不是打造完美角色。

## 出现问题时

| 现象 | 处理 |
|---|---|
| 中文显示方框 | 切换游戏字体，确认发布包解压完整 |
| 启动后立即退出 | 查看游戏目录日志，并确认没有混用旧版数据目录 |
| Android 无法安装 | 检查架构、剩余空间和系统安装权限 |
| 贴图不显示 | 确认启用贴图，且所选贴图包存在于当前版本 |
