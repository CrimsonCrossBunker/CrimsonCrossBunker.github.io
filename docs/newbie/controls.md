# 界面与操作

:::info[本课目标]
**耗时**：约 20 分钟　·　**完成标志**：能在出生点观察环境、拾取物品并查看角色状态
:::

CCB 的按键可以重新绑定，本站不假设每个版本的按键都完全相同。任何时候按 `?` 打开按键列表，按动作名称搜索，是最可靠的查询方式。

## 先掌握四类动作

### 观察

使用观察动作移动光标，查看远处格子的地形、物品和生物。进入街道或陌生建筑前先观察一圈；看见敌人时，顺便确认门、窗、围栏和撤退路线。

### 互动

交互动作负责使用门、家具和身边环境。找不到某个功能时，先查看交互菜单，而不是随机尝试字母键。

### 物品管理

- 查看物品栏，确认穿戴、携带与容器中的物品。
- 用高级物品管理在地面、载具和角色容器之间批量移动物品。
- 阅读物品说明中的重量、体积、保质期、覆盖和工具品质。

### 角色状态

角色界面会显示疼痛、疲劳、口渴、体温和身体部位。状态变化通常比生命值更早提示危险。

## 在避难所做一次练习

1. 观察房间内每一种家具。
2. 拾取一个容器和一件可作为武器的物品。
3. 打开物品栏，阅读它们的详细说明。
4. 走到门边，先观察门外，再决定是否出去。
5. 打开按键帮助，搜索“等待”“奔跑”“安全模式”。

:::tip[安全模式不是打扰]
安全模式检测到威胁时会阻止无意移动。新手应保留它；提示出现后先停下来观察，而不是立刻关闭。
:::

## 常见误区

- **只看画面不看消息记录**：许多失败原因会先出现在日志里。
- **一直奔跑**：奔跑消耗体力，应留给脱离危险。
- **拿到就穿**：衣物可能增加累赘或降低速度，先比较属性。
- **记死按键**：不同键位方案可能变化，优先搜索动作名称。
