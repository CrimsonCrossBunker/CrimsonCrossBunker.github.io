# 新人教程

:::info[路线档案]
**适合**：第一次接触 CCB 或 CDDA 的玩家　·　**前置**：无　·　**最终成果**：建立一个能继续发展的安全据点
:::

CCB 不是一款要求你背下所有按键才准开始的游戏。你只需要先掌握一小组动作，在每次失败后多理解一个系统。

## 学习路线

| 阶段 | 你会完成什么 | 完成标志 |
|---|---|---|
| [准备开始](./getting-started) | 下载、安装、设置语言与贴图 | 成功进入主菜单 |
| [界面与操作](./controls) | 移动、观察、交互、管理物品 | 能在避难所完成一次搜刮 |
| [活过第一天](./first-day) | 处理水、食物、伤口与威胁 | 安全度过第一个夜晚 |
| [稳定第一周](./first-week) | 建立据点、工具链与探索节奏 | 有稳定饮水、食物和撤退路线 |

## CCB 是什么

Cataclysm: Cleanwater Bomb（大灾变：净化协议）是基于 CDDA 持续开发的社区分支。它保留开放世界、复杂物品和自由生存玩法，同时维护自己的功能、平台构建、翻译与贴图生态。

这意味着两件事：

1. 大部分 CDDA 基础知识同样适用。
2. 下载、问题反馈和贡献应优先以 CCB 仓库及本站为准。

## 新手的三个原则

- **信息比击杀重要**：先用观察功能确认敌人、地形和退路。
- **一次只解决一个问题**：口渴就找容器和净水方式，受伤就先止血。
- **随时允许撤退**：拉开距离、关门、翻窗和绕建筑都比硬拼更可靠。
